# TTs_vehicle-counting-yolov5

## Run Project
installation
```
git clone https://github.com/0ju-un/TTs_vehicle-counting-yolov5.git
pip install -r requirements.txt  # install dependencies
```

run main process
```
python main.py --yolo-weights yolov5n.pt      # bboxes only
```
